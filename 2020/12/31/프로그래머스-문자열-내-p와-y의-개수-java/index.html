문자열 내 p와 y의 개수 (Level 1)

https://programmers.co.kr/learn/courses/30/lessons/12916

문제 설명

대문자와 소문자가 섞여있는 문자열 s가 주어집니다. s에 ‘p’의 개수와 ‘y’의 개수를 비교해 같으면 True, 다르면 False를 return 하는 solution를 완성하세요. ‘p’, ‘y’ 모두 하나도 없는 경우는 항상 True를 리턴합니다. 단, 개수를 비교할 때 대문자와 소문자는 구별하지 않습니다.

예를 들어 s가 pPoooyY면 true를 return하고 Pyy라면 false를 return합니다.


제한사항

  • 문자열 s의 길이 : 50 이하의 자연수
  • 문자열 s는 알파벳으로만 이루어져 있습니다.

입출력 예



문제풀이

  • pCount와 yCount를 만들어서 각각 문자를 셀 수 있도록 하는 변수 생성
  • 문자열의 길이만큼 for문을 돌면서 p, y가 각각 일치할 경우 카운트 증가
  • charAt() 메서드를 이용해 문자열에서 i 자리에 있는 문자를 뽑아 p와 y 를 찾는다
  • 최종으로 개수를 비교해서 같으면 true, 다르면 false 를 반환

다른사람의 풀이

  • 람다와 스트림을 사용한 풀이
  • toUpperCase() 를 이용해 문자열을 모두 대문자로 변경
  • 스트링의 각 문자를 CharSequence에 정의된 chars()를 이용해서 IntStream으로 변환( = 아스키 코드로 변환)
  • filter()를 이용해서 filter() 메서드의 매개변수 e가 p나 y의 개수와 같은지 확인해서 반환
  • 문자열의 대/소문자를 구별하지 않는다고 해서 그런지, toUpperCase() 나 toLowerCase() 를 사용해 푼 풀이가 많았다.
    • 대/소문자 구분이 필요하지 않을 때 두고두고 잘 쓰일테니 기억하자
