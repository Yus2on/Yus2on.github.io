두 개 뽑아서 더하기 (Level 1)

https://programmers.co.kr/learn/courses/30/lessons/68644


문제 설명

정수 배열 numbers가 주어집니다. numbers에서 서로 다른 인덱스에 있는 두 개의 수를 뽑아 더해서 만들 수 있는 모든 수를 배열에 오름차순으로 담아 return 하도록 solution 함수를 완성해주세요.


제한사항

  • numbers의 길이는 2 이상 100 이하입니다.
    • numbers의 모든 수는 0 이상 100 이하입니다.

입출력 예


문제 요약

  • 정수 배열 : numbers
  • numbers 배열의 각 인덱스에서 두 개의 수로 만들 수 있는 합을 모두 구한 후 오름차순으로 return


문제풀이

  • ArrayList를 사용하여 중복 값을 넣지 않도록 한 후, 값들을 array에 담고 정렬 한다.
  • 인덱스 값을 각각 더해주기 위해 이중 반복문 사용
    • for문을 돌면서 각각 값을 더해준다
    • 중복된 합의 값이 있을 수 있으니까 합을 따로 저장 ( = list)
  • 배열에 담긴 값 소트 후 반환



다른 사람의 풀이

HashSet을 이용하여 푸는 방법


HashSet

  • Set의 대표적인 구현체

  • 중복된 값을 허용하지 않음

  • 순서가 없는 데이터의 집합을 다룸

  • null 값 저장 가능

  • 내부적으로 HashMap 을 사용하여 데이터를 저장
