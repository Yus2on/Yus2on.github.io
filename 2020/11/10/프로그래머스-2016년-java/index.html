2016년 (Level 1)

https://programmers.co.kr/learn/courses/30/lessons/12901

문제 설명

2016년 1월 1일은 금요일입니다. 2016년 a월 b일은 무슨 요일일까요? 두 수 a ,b를 입력받아 2016년 a월 b일이 무슨 요일인지 리턴하는 함수, solution을 완성하세요. 요일의 이름은 일요일부터 토요일까지 각각 SUN,MON,TUE,WED,THU,FRI,SAT 입니다.

예를 들어 a=5, b=24라면 5월 24일은 화요일이므로 문자열 TUE를 반환하세요.


제한사항

  • 2016년은 윤년입니다.
  • 2016년 a월 b일은 실제로 있는 날입니다. (13월 26일이나 2월 45일같은 날짜는 주어지지 않습니다)

입출력 예

문제풀이

  • 각 월의 최대값(= 마지막 날짜)과 요일을 각각 배열로 구성한다.
  • a 월 이전까지의 일, a 월의 현재 일수를 구한다.
  • 전체 일수 % 7을 해서 요일을 구한다. (나머지 0 == 일요일)

문제점

  • test13을 통과하지 못함
  • 1월 1일이 금요일이니까 배열 인덱스 순서도 1번 인덱스에 금요일이 오도록 한다
  • 7로 나눈 나머지는 0 ~ 6 까지만….
  • 1월 1일부터 a월 b일까지 걸리는 일수를 구하고 7로 나머지 연산이 필요하다.

다른 사람의 풀이

  • calendar 클래스를 활용한 풀이이다.
  • 보다 짧게 코드를 구현했는데, calendar에 저런 함수가 있는 줄 몰랐다…

Calendar

  • Calendar 는 추상 클래스 이기 때문에 직접 객체(= 인스턴스) 를 생성할 수 없다
  • getInstance static 메서드를 이용해서 객체 생성 필요
  • 설정된 매개변수들로 만들어진 Calendar 를 반환
  • 타입 매개변수를 type 로 설정
  • 날짜 매개변수들을 year, month, doyOfMonth 로 설정
  • 지정된 스타일 및 로컬에서 매개변수로 전달 된 달력 필드 값의 문자열 표현을 반환
  • 사용가능한 문자열 표현이 없는 경우 null 을 반환한다.
