수박수박수박수박수박수? (Level 1)

https://programmers.co.kr/learn/courses/30/lessons/12922

문제 설명

길이가 n이고, 수박수박수박수….와 같은 패턴을 유지하는 문자열을 리턴하는 함수, solution을 완성하세요. 예를들어 n이 4이면 수박수박을 리턴하고 3이라면 수박수를 리턴하면 됩니다.

제한사항

  • n은 길이 10,000이하인 자연수입니다.

입출력 예

문제풀이

  • ‘수’ / ‘박’ 이 반복되기 때문에 반복문을 사용
  • 어떤 식으로 수 / 박을 체크할 것인가?
    1. case 1 : n이 홀수일 때 수, 짝수일 때는 박 출력
    2. case 2 : n만큼 반복 시 기존 문자열의 맨 마지막이 수 / 박인지 체크 필요 -> 맨 마지막 인덱스 반환

다른 사람의 풀이

  • 이렇게 풀 생각은 하지도 못 했다..
  • 리플에 가장 빠른 코드라는 말이 보여 궁금해서 실행해봤다
  • 좋은 코드일까? 코드만 봐도 어떤 의미인지 한번에 알 수는 있는 것 같다.

  • 이런 풀이도 있었다.
    1. new String(new char [n / 2 + 1]) -> 길이가 n / 2 + 1인 String 생성
    2. String 초기화를 char 배열로 해서 각 자리에 null 이 들어감
    3. replace("\0", "수박") -> 각 자리마다 “수박” 으로 치환(ex. 3자리 문자열일 때 “수박수박수박”)
    4. 원하는 문자열이 “수박수” 일 수도 있기 때문에 substring 메서드를 사용해서 잘라내서 반환
