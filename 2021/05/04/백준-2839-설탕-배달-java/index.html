[백준] 2839 / 설탕 배달 / Java

TIL (Today I Learned)

설탕 배달

https://www.acmicpc.net/problem/2839


문제 설명

  • 상근이는 요즘 설탕공장에서 설탕을 배달하고 있다. 상근이는 지금 사탕가게에 설탕을 정확하게 N킬로그램을 배달해야 한다. 설탕공장에서 만드는 설탕은 봉지에 담겨져 있다. 봉지는 3킬로그램 봉지와 5킬로그램 봉지가 있다.

  • 상근이는 귀찮기 때문에, 최대한 적은 봉지를 들고 가려고 한다. 예를 들어, 18킬로그램 설탕을 배달해야 할 때, 3킬로그램 봉지 6개를 가져가도 되지만, 5킬로그램 3개와 3킬로그램 1개를 배달하면, 더 적은 개수의 봉지를 배달할 수 있다.

  • 상근이가 설탕을 정확하게 N킬로그램 배달해야 할 때, 봉지 몇 개를 가져가면 되는지 그 수를 구하는 프로그램을 작성하시오.



입력

  • 첫째 줄에 N이 주어진다. (3 ≤ N ≤ 5000)

출력

  • 상근이가 배달하는 봉지의 최소 개수를 출력한다. 만약, 정확하게 N킬로그램을 만들 수 없다면 -1을 출력한다.

입출력 예

입력 출력
18 4
4 -1
6 2
9 3
11 3


문제풀이

  • 5로 나누어 떨어지면 몫을 sugar 에 더해준다.
  • 그렇지 않을 땐 n에서 3씩 빼서 0이 될 때 다시 첫번째 if문 반복
  • n에서 3씩 빼서 음수가 나오면 -1 로 종료.

정리

  • 봉지의 개수를 최소화하려면 최대한 5kg 봉지로 구성해야한다.
  • 그래서 먼저 최대한 5로 나눠보고 안 된다면 수를 줄여나가는 식이다.
  • 11kg의 경우에도 5로 나누면 나머지가 1이지만 실제로는 5, 3, 3 으로 구성이 가능해서 몫에서 하나씩 빼고 3으로 나눠질 수 있는지를 검사해야 했다.
  • 그래서 else 문에 5로 안 나누어지고 나머지를 3으로 나눌 수 있는 경우를 넣었다.
  • 그 아래 if문은 실상 이도저도 아닌 경우로 0보다 작으면 5나 3이나 나눠질 수 없는 값이라 -1 출력하고 반복문을 종료한다.
