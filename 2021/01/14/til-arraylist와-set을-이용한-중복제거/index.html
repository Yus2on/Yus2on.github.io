[TIL] ArrayList와 Set을 이용한 중복제거

Today I Learned (배열, set 중복제거)



1. 배열의 요소를 확인하고 중복 제거

contains()

  • boolean contains(CharSequence s)

  • contains() 은 대상 문자열에 특정 문자열이 포함되어 있는지 확인하는 함수

  • 대/소문자와 공백을 구분한다


2. Set을 이용한 중복제거 (HashSet, LinkedHashSet, TreeSet)

  • Set은 요소의 중복을 허용하지 않는 데이터의 집합

  • 단순히 데이터들을 add 하는 것으로 중복을 제거할 수 있어서 효율적

  • Set의 하위 클래스로는 HashSet, LinkedHashSet, TreeSet 등이 존재

    • HashSet : 데이터의 순서를 전혀 보장하지 않음
    • LinkedHashSet : 추가된 순서에 따라 저장
    • TreeSet : 자동으로 정렬(default 오름차순) 후 저장

  • 순서가 보장되어야 하는 경우에는 LinkedHashSet , 정렬이 필요한 경우 TreeSet 사용
