문제 설명

  • 두 정수 A와 B를 입력받은 다음, A+B를 출력하는 프로그램을 작성하시오.

입력

  • 입력은 여러 개의 테스트 케이스로 이루어져 있다.
  • 각 테스트 케이스는 한 줄로 이루어져 있으며, 각 줄에 A와 B가 주어진다. (0 < A, B < 10)

출력

  • 각 테스트 케이스마다 A+B를 출력한다.

입력 출력
1 1 2
2 3 5

9 8 17

A + B (6)

https://www.acmicpc.net/problem/10952

문제 설명

  • 두 정수 A와 B를 입력받은 다음, A+B를 출력하는 프로그램을 작성하시오.

입력

  • 첫째 줄에 테스트 케이스의 개수 T가 주어진다.
  • 각 테스트 케이스는 한 줄로 이루어져 있으며, 각 줄에 A와 B가 주어진다. A와 B는 콤마(,)로 구분되어 있다. (0 < A, B < 10)

출력

  • 각 테스트 케이스마다 A+B를 출력한다.

입출력 예

입력 출력
5 테스트 케이스 개수
1,1 2
2,3 5
3,4 7
9,8 17
5,2 7


문제풀이


정리

  • 모두 한자리수의 정수 두 개를 받아 더하는 문제들이다. A + B (6) 이후로는 더하기에 공통 문자열도 추가로 출력해줘야 해서 일단 게시글을 나누었다.
  • Scanner 대신 BufferedReader 를 사용했다.
  • readLine() 은 한 행을 전부 읽기 때문에 공백( )이나 콤마(,) 로 들어오는 문제는 StringTokenizer 를 사용해 문자열을 분리했다.
    • 구분된 변수를 꺼낼 때는 차례대로 nextToken(); 을 해주면 문자열을 반환해준다.
    • 이때 반환시킨 문자열은 반환됨과 동시에 해당 객체에서 사라지게 된다.
  • 반환한 문자들은 전부 Integer.parseInt()로 int 형으로 변환시켰다.
  • 테스트 케이스 개수가 있는 경우에는 Integer.parseInt(br.readLine()); 을 이용해 첫번째 줄의 정수를 변수에 저장하고, 그만큼 반복문을 돌렸다.
  • 없는 경우에는 while 문을 사용해 입력을 받을 때까지 반복했다.
  • 출력에서 StringBuilder 를 쓰기도 했는데, 한 줄에 A + B 를 출력하기 위해 사용했다.
  • A + B (5) 의 경우 0 두 개를 입력받은 시점에 반복문을 종료해야 해서, while 문을 true 로 무한 반복 시켜놓고, nextToken() 으로 반환된 정수가 0 이면 break 를 걸었다.
