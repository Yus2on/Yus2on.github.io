그대로 출력하기 (1)

https://www.acmicpc.net/problem/11718


문제 설명

  • 입력 받은 대로 출력하는 프로그램을 작성하시오.


입력

  • 입력이 주어진다. 입력은 최대 100줄로 이루어져 있고, 알파벳 소문자, 대문자, 공백, 숫자로만 이루어져 있다.
  • 각 줄은 100글자를 넘지 않으며, 빈 줄은 주어지지 않는다. 또, 각 줄은 공백으로 시작하지 않고, 공백으로 끝나지 않는다.

출력

문제 설명

  • 입력 받은 대로 출력하는 프로그램을 작성하시오.


입력

  • 입력이 주어진다. 입력은 최대 100줄로 이루어져 있고, 알파벳 소문자, 대문자, 공백, 숫자로만 이루어져 있다.
  • 각 줄은 100글자를 넘지 않으며, 빈 줄이 주어질 수도 있고, 각 줄의 앞 뒤에 공백이 있을 수도 있다.

출력

  • 입력받은 그대로 출력한다.

정리

  • 공백을 포함해 입력 받은 대로 출력하는 문제이다.
  • 이전 문제들과 마찬가지로 Scanner 대신 BufferedReader 를 사용했고, 11718번 문제는 StringBuilder, 11719번 문제는 BufferedWriter 를 사용했다.
  • 두 문제 다 테스트 케이스 개수가 따로 존재하지 않아서 입력 받은 문자열을 문자열 변수에 넣어주고 null인지 확인을 하여 반복문을 돌렸다.
