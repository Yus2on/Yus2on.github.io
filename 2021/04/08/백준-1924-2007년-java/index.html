[백준] 1924 / 2007년 / Java

TIL (Today I Learned)

2007년

https://www.acmicpc.net/problem/1924


문제 설명

  • 오늘은 2007년 1월 1일 월요일이다. 그렇다면 2007년 x월 y일은 무슨 요일일까? 이를 알아내는 프로그램을 작성하시오.


입력

  • 첫째 줄에 빈 칸을 사이에 두고 x(1 ≤ x ≤ 12)와 y(1 ≤ y ≤ 31)이 주어진다. 참고로 2007년에는 1, 3, 5, 7, 8, 10, 12월은 31일까지, 4, 6, 9, 11월은 30일까지, 2월은 28일까지 있다.

출력

  • 첫째 줄에 x월 y일이 무슨 요일인지에 따라 SUN, MON, TUE, WED, THU, FRI, SAT중 하나를 출력한다.

입출력 예

입력 출력
1 1 MON
3 14 WED
9 2 SUN
12 25 TUE


문제풀이

  • x월 y일을 입력받아 1월1일과 총 몇일이 차이나는지 계산을 먼저 한다. 나는 x 와 y 를 각각 month 와 day 라는 변수로 지정했다.
  • BufferedReader와 StringTokenizer 를 사용해 x와 y를 입력받는다.
  • 계산방법은 x-1월까지 그 달의 총 일수에 y(day)를 더한다.
  • 차이를 구하고 나면 그 값을 7로 나눈다. 7로 나누면 나머지가 0 ~ 6 이 생길텐데 순서대로 일요일, 월요일 ~ 토요일까지 매칭이 된다. ( = 문제에서 2007년 1월 1일은 일요일)

정리

  • 처음에는 if 문으로 day % 7 == 0 식으로 하나하나 쓰다가 너무 코드가 보기 어려워져서 switch문으로 변경하였다. 다음에는 두 번 작업하는 것을 줄여야 겠다.
